About ASCII

ASCII (American Standard Code for Information Interchange) is a character encoding standard for electronic communication. ASCII codes represent text in computers, telecommunications equipment, and other devices.

ASCII Character Sets

  • Standard ASCII - Uses 7 bits to represent 128 characters (0-127)
  • Extended ASCII - Uses 8 bits to represent 256 characters (0-255)

ASCII Categories

  • Control Characters (0-31) - Non-printable characters used for controlling devices
  • Printable Characters (32-126) - Letters, digits, punctuation, and symbols
  • Extended Characters (128-255) - Additional symbols, graphics, and international characters

Common ASCII Control Characters

NUL (0) Null character
BEL (7) Bell
BS (8) Backspace
HT (9) Horizontal Tab
LF (10) Line Feed
CR (13) Carriage Return
ESC (27) Escape
DEL (127) Delete
